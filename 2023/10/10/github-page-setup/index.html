github-page-setup

prerequest

一个github账号
nodejs >= 14

start

1.建立一个github仓库

建立仓库
建议仓库命名为<username>.github.io (e.g.) tabbleman.github.io
然后把这个url复制下来

2.下载hexo-cli

前提是已经装好了nodejs, 可以通过node --version 或者 npm --version查看
npm install hexo-cli -g
如果安装没问题那么就可以通过hexo -v 查看hexo安装是否没问题。

3.建立写blog的文件夹

hexo init blog用hexo初始化blog文件夹
然后进入文件夹blog, 输入 npm install

根据这篇blog, 需要更改_config.yml文件
以下是一些需要更改的地方

然后输入hexo clean && hexo g && hexo -d就好了

撰写blog

hexo new page-name就能建立一个新的blog了。
然后编辑source/_posts/page-name.md就能书写blog了
